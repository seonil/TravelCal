# TravelCal

여행 경비 정산 웹 애플리케이션

## 소개

TravelCal은 여행 중 발생한 경비를 간편하게 정산할 수 있는 웹 앱입니다. 누가 얼마를 썼는지 기록하면, 누가 누구에게 얼마를 보내야 하는지 자동으로 계산해줍니다.

## 주요 기능

- 참가자 등록
- 지출 내역 기록 (결제자, 금액, 설명)
- 자동 정산 계산 (최소 송금 횟수로 최적화)
- 정산 결과 클립보드 복사 (카카오톡 등에 바로 붙여넣기)
- 브라우저 로컬 스토리지에 데이터 자동 저장

## 사용 방법

1. 여행 참가자 이름 입력
2. 지출 내역 추가 (누가 얼마를 결제했는지)
3. 정산 결과 확인 및 공유
